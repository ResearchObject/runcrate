Usage

runcrate

runcrate [OPTIONS] COMMAND [ARGS]...

convert

Convert a CWLProv RO bundle into a Workflow Run RO-Crate.

RO_DIR: top-level directory of the CWLProv RO

runcrate convert [OPTIONS] RO_DIR

Options

-o, --output <output>

output RO-Crate directory or zip file

-l, --license <license>

license URL (or WorkflowHub-accepted id)

-w, --workflow-name <STRING>

original workflow name

--readme <readme>

path to a README file (should be README.md in Markdown format)

Arguments

RO_DIR

Required argument

report

Read a Workflow Run RO-Crate and report on the actions it describes.

RO_CRATE: RO-Crate directory or zip file.

runcrate report [OPTIONS] RO_CRATE

Arguments

RO_CRATE

Required argument

run

Run the workflow from a Workflow Run RO-Crate.

RO_CRATE: RO-Crate directory or zip file.

runcrate run [OPTIONS] RO_CRATE

Options

--executable <executable>

workflow runner executable

--keep-wd

keep working directory

--dry-run

do not actually run the workflow (implies –keep-wd)

Arguments

RO_CRATE

Required argument

version

Print version string and exit.

runcrate version [OPTIONS]
